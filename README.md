这是一个Go语言初学者练习项目，主要包含以下功能：

## 项目结构

- main.go: 包含主函数和随机数生成功能
  - GenRand(): 生成随机数
  - main(): 程序入口

- act/Todo.go: TODO功能实现
- model/task.go: 任务模型定义

## 使用方法

1. 安装依赖：
   ```bash
   go mod tidy
   ```

2. 运行程序：
   ```bash
   go run main.go
   ```
